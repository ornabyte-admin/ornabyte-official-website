CareHugs — Homecare Booking Platform for Kerala

Category

Client

Tech Stack

Project Overview

CareHugs is a modern homecare platform designed to connect clients across Kerala with trusted providers for essential services — from cooking and cleaning to babysitting and elderly care. Built as a scalable MVP, the platform enables real-time bookings, secure payments, and seamless caregiver-client interaction.

Developed with Django, PostgreSQL, and Razorpay, CareHugs is engineered for reliability, privacy, and performance — featuring dashboards for both clients and service providers.

  • Real-time booking system with request/accept workflow
  • Instant notifications using Django Channels + WebSockets
  • Razorpay integration with post-service payment release
  • Secure client/provider authentication (JWT)
  • Responsive, minimal UI with separate dashboards

Building a Human-Centered MVP — Byte by Byte

CareHugs wasn't just a platform; it was a promise of help delivered with tech. We created clean, distraction-free dashboards, ensuring both clients and providers could focus on what mattered — care. Every click, screen, and service was optimized for ease, speed, and trust.

We started by mapping the needs of both clients and providers — defining clear user flows, edge cases, and real-world booking behaviors in Kerala's service landscape.

Using clean UI principles, we designed separate dashboards with intuitive workflows, backed by a scalable Django architecture divided into users, bookings, services, and notifications modules.

We implemented JWT-based login, real-time WebSocket notifications, and Razorpay payment logic that holds funds until service completion confirmation by both parties.

Before release, the platform was tested in both mobile and desktop scenarios with test users — to ensure flawless booking flow, secure payments, and clear communication across roles.
