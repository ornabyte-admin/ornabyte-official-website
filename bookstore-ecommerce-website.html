Project Overview

This project involved building a full-stack online bookstore platform for selling books across multiple categories. Designed with a clean UI and robust backend architecture, the ecommerce system includes product listings, secure authentication, cart functionality, and integrated Razorpay/PayPal payments. Deployed on AWS, the platform ensures fast, scalable performance for Indian readers and sellers alike.

  • Multi-category product listing for fiction, academic, self-help, etc.
  • Secure login and user authentication
  • Add to cart, wishlist, and dynamic checkout flow
  • Payment gateway integrations: Razorpay & PayPal
  • Responsive design optimized for mobile users
  • Hosted on AWS for speed, scale, and uptime

From Shelf to Screen — Seamless Reader Journey

We focused on user flow, discoverability, and performance to ensure every click brings the reader closer to a book they love — from homepage to checkout.

We evaluated the target user personas — casual readers, students, and category-focused shoppers — and mapped out core ecommerce flows.

We used Django with PostgreSQL to create a scalable backend — managing books, users, payments, and orders with precision and flexibility.

Our design focused on clean visuals, genre browsing, and mobile usability to enhance the entire reader journey.

Razorpay and PayPal were integrated securely, and the entire system was deployed on AWS for maximum availability and speed.
